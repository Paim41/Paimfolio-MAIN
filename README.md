# PaimFolio

A modern, responsive portfolio website designed to present projects, technical skills, and creative development work in a clean and structured interface.

**Live Demo:** https://www.paimfolio.my.id/

---

## Overview

PaimFolio serves as a personal developer portfolio, focusing on clarity, performance, and user experience. It provides a centralized platform to showcase selected projects, highlight technical expertise, and establish a professional online presence.

---

## Features

- Structured project showcase
- Skills and technology overview
- Responsive layout for all screen sizes
- Clean and minimal user interface
- Smooth transitions and visual enhancements
- Optimized performance and loading speed

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Styling:** Custom CSS / Utility-first framework (if applicable)  
- **Deployment:** Vercel with custom domain
